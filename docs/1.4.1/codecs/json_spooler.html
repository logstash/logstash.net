json_spooler

Milestone: 0

Synopsis

This is what it might look like in your config file: 
# with an input plugin:
# you can also use this codec with an output.
input { 
  file {
    codec => json_spooler {
      spool_size => ... # number (optional), default: 50
    }
  }
}

Details

spool_size

  • Value type is number
  • Default value is 50
