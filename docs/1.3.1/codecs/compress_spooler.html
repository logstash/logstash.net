compress_spooler

Milestone: 1

Synopsis

This is what it might look like in your config file: 
# with an input plugin:
# you can also use this codec with an output.
input { 
  file { 
    codec =>   compress_spooler {
      compress_level => ... # number (optional), default: 6
      spool_size => ... # number (optional), default: 50
  }
  }
}

Details

compress_level

  • Value type is number
  • Default value is 6

spool_size

  • Value type is number
  • Default value is 50
